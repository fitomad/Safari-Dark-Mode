Prueba de Dark Mode

Si tienes activado el Modo Oscuro en tu macOS y estás viendo esta página en macOS Mojave son Safari Technology Preview 68 verás esta página con el fondo negro y el texto en blanco.

Si tienes macOS Mojave con el Modo Oscuro desactivado o no estás usando Safari Technology Preview el fondo de la página será blanco y el texto oscuro.

El Código

La magia está en prefers-color-scheme, que es la nueva opción para Media Query añadida a Safari Technology Preview 68
